# 项目说明：无人机可视化大屏（Vite+React）

## 1. 项目定位
这是一个前端可视化项目（Vite + React + Tailwind），用于展示/承载无人机相关的仪表盘与页面结构，偏“前端重构/展示层模板”。

## 2. 当前状态
- 可独立运行的前端工程。
- 目前未看到与后端强绑定的启动要求（后续可对接任意 API/WS）。

## 3. 目录结构速览（关键内容）
- `src/`：React 业务代码。
- `src/components/DroneDashboard/`：大屏相关组件（按命名推断）。
- `src/pages/Home.tsx`：页面入口之一。
- `tailwind.config.js` / `vite.config.ts`：构建与样式配置。

## 4. 快速运行（Windows）
1) 安装依赖：`pnpm install`
2) 启动开发服务：`pnpm run dev`
3) 访问：`http://localhost:3000`

> 归档提示：本仓库已清理 `node_modules/`（可再生成），首次运行请先执行 `pnpm install`。
